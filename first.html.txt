1) git init: generate the folder .git (hidden)
2) Create new file say "index.html" and write some content
3) git status --- untracked file
4) git add -A --- send the file in "staging area"
5) git status --- changes to be committed
6) git commit -am "any meaningful comment"

7) git push -u origin main
 > here origin is the alias name of github repo url
 > main is the branch of repo
